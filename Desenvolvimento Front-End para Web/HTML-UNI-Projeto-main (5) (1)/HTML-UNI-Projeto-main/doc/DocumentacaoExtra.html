DOCUMENTAÇÃO CSS - MERAKI VITALSYNC

Sistema de Estilização Unificado

Visão Geral

Este documento descreve o sistema CSS unificado para o website Meraki VitalSync. O stylesheet consolidado combina as melhores práticas de todos os arquivos CSS individuais em um sistema de design coeso.

Esquema de Cores

Tema Principal: Azul & Whitesmoke

Variáveis de Cor

Variável CSS Cor Uso
--primary-blue #4a90e2 Cor principal da marca, botões, bordas
--primary-dark #151b3d Fundos escuros, texto principal
--secondary-blue #3a7bc8 Estados hover, acentos
--bg-light whitesmoke Fundo da página
--bg-white #ffffff Fundo de seções
--bg-section #f8f9fa Fundo de artigos

Estrutura de Arquivos

Arquivos Atuais

  • style-unified.css - Stylesheet consolidado (RECOMENDADO)
  • style.css - Stylesheet base original
  • style-old-backup.css - Backup do style.css original
  • Arquivos CSS individuais (home.css, contato.css, etc.) - Podem ser descontinuados

Breakpoints Responsivos

Tablet (max-width: 992px)

  • Navegação torna-se vertical
  • Tamanhos de fonte reduzidos
  • Layouts de coluna única
  • Círculos numerados menores em listas ordenadas

Mobile (max-width: 600px)

  • Header torna-se vertical
  • Links de navegação menores
  • Tamanho de texto hero reduzido
  • Imagens de logo menores

Guia de Migração

Passo 1: Backup

Todos os arquivos CSS originais foram preservados. O style.css original está em backup como style-old-backup.css.

Passo 2: Atualizar Arquivos HTML

Altere todos os arquivos HTML para usar o stylesheet unificado:

<!-- DE: -->
<link rel="stylesheet" href="style/home.css">

<!-- PARA: -->
<link rel="stylesheet" href="style/style-unified.css">

Passo 3: Testar Cada Página

  1. Abrir cada arquivo HTML em um navegador
  2. Verificar layout, cores e interações
  3. Testar comportamento responsivo
  4. Verificar se formulários e botões funcionam corretamente
